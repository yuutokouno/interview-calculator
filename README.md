# 電卓アプリ実装課題

## 課題概要

このリポジトリには、電卓アプリのベースとなる `index.html` を用意しています。
HTML と CSS で画面の見た目はすでにできあがっているので、ご安心ください！

**やっていただきたいことは、JavaScript を追加して電卓として動くようにすることです。**

ボタンを押すと画面に数値や演算子が表示され、計算ができる状態を目指しましょう。

> **💡 補足：** リポジトリ内に `e2e/` フォルダや `playwright.config.js`、`package.json`、`.github/` フォルダがありますが、これらは採点用の自動テスト環境です。**中身を気にする必要はありません。** `index.html` だけに集中してください。

---

## 必須要件

以下の機能を実装してみてください。

- 数字ボタン（0〜9）を押すと、画面（表示エリア）に反映される
- 四則演算（`+`, `-`, `×`, `/`）が正しく動作する
- かっこ `()` を使った計算ができる
- クリアボタン（C）で入力がリセットされる
- 1文字削除ボタン（←）で直前の入力を削除できる
- イコールボタン（=）で計算結果が表示される

## 提出方法

### Step 1：テンプレートから自分のリポジトリを作る

1. このリポジトリページ上部の **「Use this template」** ボタンから **「Create a new repository」** を選びます
2. リポジトリ名は自由に設定してOKです
3. **必ず Public** を選択してください（Private だと提出先から確認できません）
4. **「Create repository」** をクリックすれば完了です

### Step 2：リポジトリをクローンして実装する

1. 作成したリポジトリをクローンして、ローカルで作業します
2. `index.html` をエディタで開き、JavaScript を追加して電卓の機能を実装してください
3. ブラウザで `index.html` を開けば動作確認できます

### Step 3：完成したコードを GitHub に push する

実装が終わったら、変更を GitHub にアップロード（push）しましょう。

```bash
git add .
git commit -m "電卓の機能を実装"
git push origin main
```

### Step 4：GitHub Pages で公開する

1. GitHub 上で自分のリポジトリページを開きます
2. **Settings** → 左メニューの **Pages** を開きます
3. 「Source」で **Deploy from a branch** を選択します
4. Branch を **main**、フォルダを **/ (root)** に設定して **Save** をクリックします
5. 数分ほど待つと、ページ上部に公開 URL が表示されます

### Step 5：URL を提出する

以下の **2つの URL** を提出してください。

1. **GitHub リポジトリの URL**
2. **GitHub Pages の URL**
